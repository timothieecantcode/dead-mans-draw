# Dead Man’s Draw (C++)

A C++ implementation of the _Dead Man’s Draw_ card game with 2 players, focusing on object-oriented design, game logic, and rule handling.

---

## Overview

This project simulates the core mechanics of _Dead Man’s Draw_, a card game where players draw cards to build their loot while avoiding busting.

The implementation focuses on:

- clean class design
- accurate rule enforcement
- state management during gameplay

---

## Features

- Full game logic implementation
- Card drawing and bust mechanics
- Special card effects (e.g. Anchor, Chest, Key, etc.)
- Turn-based gameplay system
- Object-oriented design using C++

```bash
git clone https://github.com/timothieecantcode/dead-mans-draw.git
```

2. Open in Visual Studio

3. Build and run the project

---

## How to Play

- Draw cards one by one to collect loot
- Each card type has a unique effect
- Drawing duplicate suit cards causes a **bust**
